# Sagnick Paul | Portfolio

A modern, professional portfolio website for an Electrical Engineering undergraduate specializing in AI, Machine Learning, and Robotics. Built with a focus on performance, aesthetics, and interactivity.

![Portfolio Preview](public/logo.png) <!-- Replace with a real screenshot/GIF if available -->

## 🚀 Overview

This portfolio showcases my journey, research interests, and technical projects. It features a sleek glassmorphism design, smooth parallax animations, and a fully responsive interface.

### ✨ Key Features

- **Modern Tech Stack**: Built with Next.js 14, React 18, and TypeScript.
- **Glassmorphism UI**: Semi-transparent elements with backdrop blur and soft borders for a premium feel.
- **Dynamic Animations**: Smooth scroll-triggered animations and parallax effects using Framer Motion.
- **Interactive Components**: 
  - **AI Journey Timeline**: A vertical, interactive timeline of my academic and research progress.
  - **GitHub Integration**: Live contribution graph and stats.
- **Responsive Design**: Optimized for all devices, from mobile to ultra-wide monitors.
- **Dark/Light Mode**: Seamless theme switching with `next-themes`.
- **SEO Optimized**: Following best practices for search engine visibility.

## 🛠️ Built With

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Theme Management**: [next-themes](https://github.com/pacocoursey/next-themes)
- **Language**: [TypeScript](https://www.typescriptlang.org/)


## 🏁 Getting Started

### Prerequisites

- Node.js (Version 18.x or higher)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Sagnick-Paul/Sagnick-Portfolio.git
   cd Sagnick-Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
